## Traceroutes

Traceroutes provide a hop-by-hop breakdown of the Internet path network traffic follows from Cloudflare's network to a customer's network.

Refer to <a href={props.traceRoutes}>Traceroutes</a> to learn more.

## Packet captures

Packet captures allow customers to analyze the raw packet data they send to and receive from Cloudflare's network.

Refer to [packet captures](/magic-firewall/packet-captures/) to learn more.

## Query analytics with GraphQL
