<TabItem label="macOS">

On macOS, run `ifconfig`. When WARP is turned on, you will see a `utun` interface with your WARP device IP.

```sh
ifconfig
```


<TabItem label="Linux">

On Linux, run `ifconfig` or `ip addr`. When WARP is turned on, you will see a `utun` interface with your WARP device IP.

```sh null
ip addr
```

```sh {4} output
<redacted>
```
