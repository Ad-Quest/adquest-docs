## Инструкция по замене

1. Откройте соответствующую страницу в браузере
2. Сделайте скриншот (рекомендуется 1920x1080 или выше)
3. Оптимизируйте изображение через [TinyPNG](https://tinypng.com/)
4. Замените файл в папке `public/screenshots/crm/`
5. Обновите страницу документации для проверки

## Рекомендации

### Данные для скриншотов
- Используйте реалистичные данные (не "test", "123", "asdf")
- Показывайте типичные сценарии использования
- Скройте чувствительную информацию (реальные email, телефоны)

### Технические требования
- Формат: PNG
- Разрешение: 1920x1080 или 2560x1440
- Размер файла: < 500KB
- Тема: светлая (если не указано иное)

### Качество
- Текст должен быть читаемым
- Избегайте размытых изображений
- Убедитесь, что все элементы видны
- Закройте лишние вкладки браузера

## Проверка после замены
