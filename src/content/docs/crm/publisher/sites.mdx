# Управление сайтами

Руководство по добавлению и управлению сайтами в панели издателя.

## Добавление сайта

Чтобы добавить новый сайт:

1. Нажмите кнопку "Добавить сайт"

  src="/screenshots/crm/publisher/add-site-button.png" 
  alt="Кнопка 'Добавить сайт' в правом верхнем углу панели издателя"
  caption="Кнопка для добавления нового сайта"
/>

2. Заполните форму

<Screenshot 
  src="/screenshots/crm/publisher/add-site-form.png" 
  alt="Форма добавления сайта с полями: название, URL, описание и категория"
  caption="Форма добавления нового сайта"
/>

### Шаг 1: Основная информация

| Поле | Описание | Обязательное |
|------|----------|--------------|
| Название | Название вашего сайта | Да |
| URL | Полный адрес сайта (с https://) | Да |

### Шаг 2: Дополнительная информация

| Поле | Описание | Обязательное |
|------|----------|--------------|
| Описание | Краткое описание сайта | Нет |
| Категория | Тематика сайта | Да |

Доступные категории:
- Новости (news)
- Развлечения (entertainment)
- Технологии (technology)
- Бизнес (business)
- Образ жизни (lifestyle)
- Спорт (sports)
- Образование (education)
- Здоровье (health)
- Путешествия (travel)
- Еда (food)
- Автомобили (automotive)
- Недвижимость (real_estate)
- Игры (gaming)
- Другое (other)

### Шаг 3: Верификация домена

После создания сайта необходимо подтвердить владение доменом:

1. Скопируйте TXT запись
2. Добавьте её в DNS настройки вашего домена
3. Дождитесь проверки (обычно 1-2 часа)

После верификации вы сможете [получить Site Key](#получение-site-key) для [интеграции виджета](/crm/publisher/integration/).

:::tip[Совет]
Укажите точное описание сайта - это ускорит модерацию и поможет подобрать релевантную рекламу.
:::

## Получение Site Key

После верификации домена:

1. Откройте детали сайта
2. Скопируйте Site Key (начинается с `pk_`)
3. Используйте его для [интеграции виджета](/crm/publisher/integration/)

Подробная инструкция по интеграции доступна в разделе [Интеграция виджета](/crm/publisher/integration/).

## Статусы сайта

| Статус | Описание | Действия |
|--------|----------|----------|
| На модерации | Сайт проверяется администратором | Ожидайте |
| Активен | Сайт одобрен и работает | Можно интегрировать виджет |
| Приостановлен | Сайт временно отключен | Свяжитесь с поддержкой |
| Отклонен | Сайт не прошел модерацию | Проверьте причину отклонения |

## Редактирование сайта

Для изменения информации о сайте:

1. Откройте [список сайтов](/crm/publisher/)
2. Нажмите на иконку редактирования
3. Измените необходимые поля
4. Сохраните изменения

:::caution[Внимание]
Изменение URL сайта требует повторной [верификации домена](#добавление-сайта).
:::

## Удаление сайта

Чтобы удалить сайт:

1. Откройте детали сайта
2. Нажмите "Удалить сайт"
3. Подтвердите действие

:::caution[Важно]
Перед удалением убедитесь, что [виджет удален](/crm/publisher/integration/) с сайта. Удаление необратимо.
:::

## Частые вопросы

### Как долго проходит модерация?

Обычно 1-2 рабочих дня. В редких случаях до 5 дней.

### Почему мой сайт отклонили?

Возможные причины:
- Сайт содержит запрещенный контент
- Неверный URL или сайт недоступен
- Сайт не соответствует требованиям платформы
- Дублирование уже добавленного сайта

### Можно ли изменить Site Key?

Нет, Site Key уникален для каждого сайта. При необходимости удалите сайт и добавьте заново.
