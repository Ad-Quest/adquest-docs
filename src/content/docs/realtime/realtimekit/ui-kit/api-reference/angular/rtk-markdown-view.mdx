## Properties

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `maxLength` | `number` | ✅ | - | max length of text to render as markdown |
| `text` | `string` | ✅ | - | raw text to render as markdown |

## Usage Examples

### Basic Usage

```html
<!-- component.html -->
<rtk-markdown-view></rtk-markdown-view>
```

### With Properties

```html
<!-- component.html -->
<rtk-markdown-view
 maxLength="42"
 text="example">
</rtk-markdown-view>
```
