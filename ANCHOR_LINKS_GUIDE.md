# Руководство по якорным ссылкам

## Как работают якоря в Starlight

Starlight автоматически создает якоря (anchors) из заголовков, преобразуя их в URL-friendly формат (slug).

### Правила преобразования

Русские заголовки преобразуются в кириллические slug:

| Заголовок | Автоматический slug |
|-----------|---------------------|
| `## Добавление сайта` | `#добавление-сайта` |
| `## Получение Site Key` | `#получение-site-key` |
| `## Баланс` | `#баланс` |
| `## Запрос на вывод средств` | `#запрос-на-вывод-средств` |

## Примеры использования

### Ссылка на другую страницу с якорем
```markdown
См. [Запрос на вывод средств](/crm/publisher/finances/#запрос-на-вывод-средств)
```

### Ссылка внутри той же страницы
```markdown
См. [Получение Site Key](#получение-site-key)
```

### Inline ссылка в тексте
```markdown
После [верификации сайта](/crm/publisher/sites/#добавление-сайта) вы сможете получить Site Key.
```

### Ссылка в списке
```markdown
- [Добавьте свой первый сайт](/crm/publisher/sites/#добавление-сайта)
- [Получите Site Key](/crm/publisher/sites/#получение-site-key)
- [Встройте виджет](/crm/publisher/integration/)
```

## Проверка работы якорей

После деплоя проверьте:

1. Откройте страницу документации
2. Кликните на ссылку с якорем
3. Страница должна прокрутиться к нужному заголовку
4. URL в браузере должен содержать якорь (например: `/crm/publisher/sites/#добавление-сайта`)

## Table of Contents (Оглавление)

Starlight автоматически создает оглавление (Table of Contents) справа на странице:
- Показывает все заголовки уровня 2 и 3
- Кликабельные ссылки с якорями
- Подсвечивает текущий раздел при прокрутке

## Важные замечания

1. **Не используйте синтаксис `{#custom-id}`** - это не поддерживается в MDX
2. **Якоря создаются автоматически** из текста заголовка
3. **Кириллические якоря работают** в современных браузерах
4. **Якоря чувствительны к регистру** - используйте точное написание
5. **Пробелы заменяются на дефисы** - "Добавление сайта" → "добавление-сайта"

## Отладка проблем

### Якорь не работает
- Проверьте правильность написания (регистр, дефисы)
- Убедитесь, что заголовок существует на странице
- Проверьте, что используете кириллический slug

### Ссылка ведет не туда
- Проверьте путь к странице
- Убедитесь, что якорь соответствует заголовку
- Очистите кэш браузера

### Как узнать правильный slug?
1. Откройте страницу в браузере
2. Наведите на заголовок
3. Кликните на иконку ссылки рядом с заголовком
4. Скопируйте якорь из URL
