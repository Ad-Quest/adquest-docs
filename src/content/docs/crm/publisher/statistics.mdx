Руководство по просмотру и анализу статистики ваших сайтов.

### Основные метрики

- **Показы** - количество загрузок виджета
- **Клики** - количество переходов по рекламе
- **CTR** - процент кликов (клики / показы × 100%)
- **Доход** - заработанная сумма
- **RPM** - доход за 1000 показов (Revenue Per Mille)

### Графики

- График доходов по дням
- График показов и кликов
- Распределение по сайтам
- Динамика CTR

- Текущий месяц
- Прошлый месяц
- Произвольный период

## Статистика по сайтам

Для каждого сайта доступна детальная статистика:

1. Откройте [список сайтов](/crm/publisher/)
2. Нажмите на название сайта
3. Просмотрите метрики

Доступная информация:
- Показы за период
- Клики и CTR
- Доход по дням
- Топ рекламных кампаний
- География пользователей

## Экспорт данных

Экспортируйте [статистику](/crm/publisher/statistics/) для анализа:

1. Выберите период
2. Нажмите "Экспорт"
3. Выберите формат (CSV или PDF)
4. Скачайте файл

## Как увеличить доход?

- Размещайте виджет на видном месте
- Увеличьте трафик на сайт
- Используйте виджет на нескольких страницах
- Оптимизируйте пользовательский опыт

Подробнее о размещении виджета см. [Интеграция виджета](/crm/publisher/integration/).

## Частые вопросы

### Когда начисляется доход?

Доход начисляется сразу после успешного показа рекламы пользователю.

### Почему статистика не обновляется?

Статистика обновляется с задержкой до 1 часа. Если данные не появляются дольше, проверьте интеграцию виджета.

### Как рассчитывается CTR?

CTR = (Клики / Показы) × 100%

Например: 50 кликов / 1000 показов × 100% = 5% CTR
