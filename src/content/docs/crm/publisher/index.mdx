# Панель издателя

Руководство по управлению сайтами, просмотру статистики и получению выплат для издателей.

## Обзор

Панель издателя предоставляет все необходимые инструменты для:

- [Добавления и управления сайтами](/crm/publisher/sites/)
- [Получения Site Key](/crm/publisher/sites/#получение-site-key) для интеграции виджета
- Просмотра [статистики](/crm/publisher/statistics/) показов и кликов
- Отслеживания доходов
- [Запроса выплат](/crm/publisher/finances/#запрос-на-вывод-средств)

## Главная страница

После входа в систему вы попадаете на главную страницу издателя.

### Статистика

На главной странице отображаются ключевые метрики:

- **Общий доход** - суммарный доход за все время
- **Активные сайты** - количество работающих сайтов
- **Показы сегодня** - количество показов виджета за текущий день
- **Средний CTR** - процент кликов от показов

### Недавние сайты

Список последних добавленных сайтов с информацией:

- Название сайта
- URL адрес
- [Статус](/crm/publisher/sites/#статусы-сайта) (активен, на модерации, приостановлен)
- Быстрые действия

### Быстрые действия

- [Добавить новый сайт](/crm/publisher/sites/#добавление-сайта)
- Перейти к [статистике](/crm/publisher/statistics/)
- Открыть [настройки](/crm/publisher/settings/)
