# Панель агентства

:::caution[В разработке]
Функционал агентства находится в активной разработке и пока недоступен. Эта страница будет обновлена после завершения разработки.
:::

## Планируемый функционал

После завершения разработки панель агентства будет предоставлять инструменты для:

- Управления клиентами (издателями и рекламодателями)
- Настройки комиссионных
- Просмотра агрегированной статистики
- Создания отчетов для клиентов
- Управления доступами

Следите за обновлениями на нашем сайте.
