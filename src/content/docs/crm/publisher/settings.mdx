# Настройки

Управление профилем, организацией и уведомлениями.

## Профиль

В разделе "Настройки" → "Профиль":

- Имя и фамилия
- Email адрес
- Номер телефона
- Аватар

### Изменение данных

1. Перейдите в "Настройки" → "Профиль"
2. Измените необходимые поля
3. Нажмите "Сохранить изменения"

## Организация

Информация о вашей организации:

- Название организации
- ИНН и ОГРН
- Юридический адрес
- Банковские реквизиты

:::tip[Важно]
Заполните реквизиты организации для [получения выплат](/crm/publisher/finances/#запрос-на-вывод-средств). Подробнее в разделе [Настройки → Организация](/crm/publisher/settings/#организация).
:::

### Обязательные поля

| Поле | Описание |
|------|----------|
| Название | Полное название организации |
| ИНН | Идентификационный номер налогоплательщика |
| Юридический адрес | Адрес регистрации организации |

### Банковские реквизиты

Для получения выплат укажите:

- Название банка
- БИК
- Корреспондентский счет
- Расчетный счет

## Безопасность

### Смена пароля

1. Перейдите в "Настройки" → "Безопасность"
2. Введите текущий пароль
3. Введите новый пароль
4. Подтвердите новый пароль
5. Сохраните изменения

Требования к паролю:
- Минимум 8 символов
- Хотя бы одна заглавная буква
- Хотя бы одна цифра
- Хотя бы один специальный символ

### Двухфакторная аутентификация

:::note[В разработке]
Функция двухфакторной аутентификации будет доступна в следующих версиях.
:::

## Уведомления

Настройте, какие уведомления вы хотите получать:

- Новые показы (ежедневная сводка)
- Достижение порога вывода
- Статус модерации сайтов
- Изменение статуса выплат
- Системные уведомления

### Каналы уведомлений

- Email (основной канал)
- Push-уведомления в браузере
- Уведомления в панели управления

## API ключи

:::note[В разработке]
Функция API ключей будет доступна в следующих версиях.
:::

Для интеграции с внешними системами вы сможете:

1. Создавать API ключи
2. Указывать права доступа
3. Управлять активными ключами

## Удаление аккаунта

:::caution[Важно]
Удаление аккаунта необратимо. Все данные будут удалены без возможности восстановления.
:::

Перед удалением:
- Выведите все средства с баланса
- Сохраните необходимые отчеты
- Удалите виджеты с ваших сайтов

Для удаления аккаунта обратитесь в панель управления.
