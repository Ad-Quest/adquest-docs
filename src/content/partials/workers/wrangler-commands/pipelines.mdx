<AnchorHeading title="`create`" slug="pipelines-create" depth={3} />

Create a new pipeline

```txt
wrangler pipelines create <PIPELINE_NAME> [OPTIONS]
```

- `PIPELINE_NAME` <Type text="string" /> <MetaInfo text="required" />
	- The name of the pipeline to create.
- `--sql` <Type text="string" /> <MetaInfo text="optional" />
	- Inline SQL query for the pipeline. Refer to [SQL reference](/pipelines/sql-reference/).
- `--sql-file` <Type text="string" /> <MetaInfo text="optional" />
	- Path to file containing SQL query for the pipeline. Refer to [SQL reference](/pipelines/sql-reference/).

<AnchorHeading title="`sinks create`" slug="pipelines-sinks-create" depth={3} />

Create a new sink

```txt
wrangler pipelines sinks create <SINK_NAME> [OPTIONS]
```

- `SINK_NAME` <Type text="string" /> <MetaInfo text="required" />
	- The name of the sink to create.
- `--type` <Type text="string" /> <MetaInfo text="required" />
	- The type of sink to create. Choices: [`r2`](/pipelines/sinks/available-sinks/r2/), [`r2-data-catalog`](/pipelines/sinks/available-sinks/r2-data-catalog/).
- `--bucket` <Type text="string" /> <MetaInfo text="required" />
	- R2 bucket name.
- `--format` <Type text="string" /> <MetaInfo text="optional" />
	- Output format. Choices: `json`, `parquet`.
- `--compression` <Type text="string" /> <MetaInfo text="optional" />
	- Compression method (parquet only). Choices: `uncompressed`, `snappy`, `gzip`, `zstd`, `lz4`.
- `--target-row-group-size` <Type text="string" /> <MetaInfo text="optional" />
	- Target row group size for parquet format (parquet only).
- `--path` <Type text="string" /> <MetaInfo text="optional" />
	- The base prefix in your bucket where data will be written (r2 sinks only).
- `--partitioning` <Type text="string" /> <MetaInfo text="optional" />
	- Time partition pattern (r2 sinks only).
- `--roll-size` <Type text="number" /> <MetaInfo text="optional" />
	- Roll file size in MB.
- `--roll-interval` <Type text="number" /> <MetaInfo text="optional" />
	- Roll file interval in seconds.
- `--access-key-id` <Type text="string" /> <MetaInfo text="optional" />
	- R2 access key ID (required for r2 sinks). Refer to [R2 sink authentication](/pipelines/sinks/available-sinks/r2/#authentication).
- `--secret-access-key` <Type text="string" /> <MetaInfo text="optional" />
	- R2 secret access key (required for r2 sinks). Refer to [R2 sink authentication](/pipelines/sinks/available-sinks/r2/#authentication).
- `--namespace` <Type text="string" /> <MetaInfo text="optional" />
	- Data catalog namespace (required for r2-data-catalog).
- `--table` <Type text="string" /> <MetaInfo text="optional" />
	- Table name within namespace (required for r2-data-catalog).
- `--catalog-token` <Type text="string" /> <MetaInfo text="optional" />
	- Authentication token for data catalog (required for r2-data-catalog sinks). Refer to [R2 Data Catalog authentication](/pipelines/sinks/available-sinks/r2-data-catalog/#authentication).
