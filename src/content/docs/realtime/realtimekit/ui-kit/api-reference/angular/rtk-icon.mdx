An icon component which accepts an svg string and renders it.

## Properties

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `icon` | `string` | ✅ | - | Icon |
| `size` | `Size1` | ✅ | - | Size |
| `variant` | `IconVariant` | ✅ | - | Icon variant |

## Usage Examples

### Basic Usage

```html
<!-- component.html -->
<rtk-icon></rtk-icon>
```

### With Properties

```html
<!-- component.html -->
<rtk-icon
 icon="example"
 size="md"
 variant="primary">
</rtk-icon>
```
