# Защита документации капчей

## Описание

Все страницы документации защищены капчей ADQuest. При первом посещении пользователь перенаправляется на страницу проверки.

## Структура

### Страница проверки
`/captcha/` - отдельная страница с виджетом капчи

### Глобальная проверка
`src/components/overrides/Head.astro` - проверка на всех страницах

## Логика работы

1. Пользователь заходит на любую страницу документации
2. Скрипт проверяет `sessionStorage.adquest_captcha_passed`
3. Если флаг не установлен - редирект на `/captcha/`
4. После успешной проверки:
   - Устанавливается флаг в sessionStorage
   - Редирект на главную страницу `/`
5. Пользователь может свободно перемещаться по документации

## Особенности

- Проверка действует в рамках одной сессии браузера
- При закрытии браузера флаг сбрасывается
- Капча не показывается повторно в той же сессии
- Простой дизайн без эффектов размытия

## Файлы

- `src/pages/captcha.astro` - страница с капчей
- `src/components/overrides/Head.astro` - глобальная проверка
