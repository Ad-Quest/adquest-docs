# AdQuest CRM - Панель управления

Веб-приложение для управления рекламной платформой AdQuest. Предоставляет инструменты для издателей, рекламодателей, агентств и администраторов.

## Для кого эта платформа

AdQuest CRM объединяет три стороны рекламного рынка:

- **Издатели** - владельцы сайтов, которые монетизируют трафик через рекламные CAPTCHA
- **Рекламодатели** - компании, размещающие рекламу в виджетах на сайтах издателей
- **Агентства** - посредники, управляющие клиентами и получающие комиссионные

## Быстрый старт

### Регистрация

1. Перейдите на [страницу регистрации](https://ad-quest.ru/auth/register)
2. Заполните форму:
   - Имя и фамилия
   - Email адрес
   - Пароль
   - Название организации
3. Подтвердите email
4. Войдите в систему

## Первые шаги для издателя

1. [Добавьте свой первый сайт](/crm/publisher/sites/#add-site)
2. [Получите Site Key](/crm/publisher/sites/#site-key) для интеграции
3. [Встройте виджет](/crm/publisher/integration/) на сайт
4. Дождитесь модерации
5. Начните получать доход и отслеживайте [статистику](/crm/publisher/statistics/)

## Основные возможности

### Для издателей (доступно)

- Добавление и управление сайтами
- Просмотр статистики показов и кликов
- Отслеживание доходов
- Запрос на вывод средств
- Интеграция виджета на сайт

### Для рекламодателей (в разработке)

- Создание рекламных кампаний
- Загрузка креативов
- Настройка таргетинга аудитории
- Управление бюджетом
- Аналитика эффективности

### Для агентств (в разработке)

- Управление клиентами
- Агрегированная статистика
- Настройка комиссионных
- Отчеты для клиентов

## Доступ к платформе

- **Вход:** [ad-quest.ru/auth/login](https://ad-quest.ru/auth/login)
- **Регистрация:** [ad-quest.ru/auth/register](https://ad-quest.ru/auth/register)

## Требования

- Современный браузер (Chrome, Firefox, Safari, Edge)
- Подключение к интернету
- Подтвержденный email адрес
- Для издателей: собственный сайт для размещения виджета
- Для рекламодателей: рекламные материалы и бюджет

## Безопасность

AdQuest CRM использует современные стандарты безопасности:

- Шифрование всех данных (HTTPS)
- Аутентификация через Zitadel OAuth2
- Защита от несанкционированного доступа
- Регулярные резервные копии данных
- Соответствие требованиям законодательства РФ
