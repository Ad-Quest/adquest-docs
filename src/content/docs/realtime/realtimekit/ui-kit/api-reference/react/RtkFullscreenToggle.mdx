A button which toggles full screen mode for any
 existing `rtk-meeting` component in the DOM.

## Properties

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `iconPack` | `IconPack` | ❌ | `defaultIconPack` | Icon pack |
| `size` | `Size` | ✅ | - | Size |
| `states` | `States` | ✅ | - | States object |
| `t` | `RtkI18n` | ❌ | `useLanguage()` | Language |
| `targetElement` | `HTMLElement` | ✅ | - | Target Element to fullscreen |
| `variant` | `ControlBarVariant` | ✅ | - | Variant |

## Usage Examples

### Basic Usage

```tsx
import { RtkFullscreenToggle } from '@cloudflare/realtimekit-react-ui';

function MyComponent() {
  return <RtkFullscreenToggle />;
}
```

### With Properties

```tsx
import { RtkFullscreenToggle } from '@cloudflare/realtimekit-react-ui';

function MyComponent() {
  return (
    <RtkFullscreenToggle
      size="md"
      targetElement={htmlelement}
      variant="button"
    />
  );
}
```
